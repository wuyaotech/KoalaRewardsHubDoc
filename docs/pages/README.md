| M17 | 券详情页 | 优惠券 | 券信息、核销码 |

## PC管理后台（16 页）

| 编号 | 页面 | 所属模块 | 可见权限 | 说明 |
|------|------|----------|----------|------|
| P01 | 登录页 | 用户与认证 | 所有人 | 手机号+密码登录，仅管理员/超管可进入 |
| P02 | 工作台首页 | 数据统计 | 管理员+超管 | 数据概览卡片+快捷操作+近期动态 |
| P03 | 会员列表页 | 会员信息 | 管理员+超管 | 搜索筛选、会员表格、操作入口 |
| P04 | 会员详情页 | 会员信息 | 管理员+超管 | 会员完整画像：资料+积分+等级+兑换+券 |
| P05 | 积分录入页 | 积分账户 | 管理员+超管 | 选会员→选积分类型→录入积分 |
| P06 | 积分临期查询页 | 积分账户 | 管理员+超管 | 即将清零积分的会员列表 |
| P07 | 等级规则配置页 | 会员等级 | 超管 | 等级名称、门槛、权益配置 |
| P08 | 商品列表页 | 商品管理 | 管理员+超管 | 商品表格、状态筛选、上下架操作 |
| P09 | 商品编辑页 | 商品管理 | 管理员+超管 | 商品信息表单（创建/编辑） |
| P10 | 商品分类管理页 | 商品管理 | 管理员+超管 | 分类的增删改 |
| P11 | 兑换记录页 | 兑换交易 | 管理员+超管 | 兑换订单查询、多维度筛选 |
| P12 | 核销工作台页 | 核销 | 管理员+超管 | 扫码/输码核销、确认核销 |
| P13 | 优惠券管理页 | 优惠券 | 管理员+超管 | 券模板管理、发放/核销统计 |
| P14 | 数据统计页 | 数据统计 | 管理员+超管 | 兑换汇总、会员概览、积分概览、商品分析 |
| P15 | 用户管理页 | 系统管理 | 超管 | 用户列表、设置/取消管理员 |
| P17 | 操作日志页 | 系统管理 | 超管 | 关键操作审计记录查询 |
| P18 | 积分规则配置页 | 系统管理 | 超管 | 积分比例、有效期等基础规则 |
