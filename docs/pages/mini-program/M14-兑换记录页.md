# M14 兑换记录页

**所属模块**：兑换交易

**触发场景**：个人中心 → 兑换记录

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 状态筛选 Tab | 全部 / 待核销 / 已完成 / 已过期 |
| 积分类型筛选 | 大RF / 小RF / 全部 |
| 订单列表 | 每条订单卡片 |

**每条订单卡片包含**：

| 字段 | 说明 |
|------|------|
| 商品信息 | 商品图片、名称、兑换数量 |
| 花费积分 | 扣减的积分数 |
| 兑换时间 | 下单时间 |
| 订单状态 | 待核销 / 已完成 / 已过期 |
| 操作按钮 | 待核销 → "查看核销码"；已完成/已过期 → "查看详情" |

## 交互说明

- 点击 "查看核销码" → 跳转 M15 核销码页
- 支持下拉加载更多
