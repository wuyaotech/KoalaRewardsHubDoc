## 三、关键流程串联

- **登录流程**：login-dialog 弹窗（微信一键登录/手机号验证码登录）→ 新用户填写信息或跳过 → 关闭弹窗留在当前页  
- **兑换流程**：M09 或 M00 → M10 → checkout-sheet → 创建订单 → M15；或 M12 → checkout-sheet → 创建订单 → M15  
- **积分查看**：M00 / M03 → M06 → M07  
- **券使用**：M16 → M17（核销码展示）
