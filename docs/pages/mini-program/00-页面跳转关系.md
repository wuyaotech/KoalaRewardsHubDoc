# 小程序端 · 页面跳转关系

本文档汇总各页面的**入口（从哪来）**与**出口（到哪去）**，便于开发与联调时对照。

---

## 合理性分析摘要

| 类型 | 说明 | 建议 |
|------|------|------|
| **Tab 与首屏** | 「积分商城」Tab 下首屏为 M00（头像+两种积分），再点击才进 M09；M09 入口不应写「底部 Tab」，以免与 M00 混淆 | 已按此区分：M09 入口为 M00 卡片/M06，不写 Tab 直进 M09 |
| **M09 出口遗漏** | M11、M12 单页写明「商城页入口/购物车图标」，但 M09 单页未写，跳转表易漏 | 已补充：M09 → M11 心愿单、M12 购物车 |
| **M03 与 M12** | M03 单页快捷入口只列「兑换记录、心愿单、我的券包」，未列「购物车」；M12 单页写「个人中心入口」 | 需产品确认：个人中心是否提供购物车入口；若提供，M03 单页建议补充 |
| **M00 出口表述** | 「底部 Tab → M09/M03/M16」易误解为点 Tab 就离开 M00 | 已改为：Tab 切换至 M03/M16；积分商城 Tab 下首屏即 M00，再可进 M09 |
| **M14 查看详情** | 待核销为「查看核销码」→ M15；已完成/已过期为「查看详情」，目标页未定 | 保持「若单独有页则补充」；若无订单详情页可做弹窗或 M15 |
| **整体** | 无孤岛页，主流程闭环清晰；返回逻辑在第四节已补全 | 实现时按第四节建议统一返回行为 |

## 一、按页面列跳转关系

| 页面 | 入口（从哪来） | 出口（到哪去） |
|------|----------------|----------------|
| **M00 H5首页** | 微信授权+手机号绑定完成后；底部 Tab「积分商城」时首屏 | 头像/会员标识 → M03 或 M08；积分 → M06 / M07；大RF/小RF卡片 → M09；底部 Tab 切换 → M03 会员中心 / M16 我的券包（积分商城 Tab 下首屏即本页，可再进 M09） |
| **M01 微信授权登录页** | 首次打开 / 登录态过期 | 已绑定手机号 → 首页（M00）；未绑定 → M02 手机号绑定页 |
| **M02 手机号绑定页** | M01 授权成功后未绑定手机号 | 绑定成功 → M03 个人中心 或 首页（M00） |
| **M03 个人中心页** | 底部 Tab「会员中心」；M00 头像区点击 | 会员卡片 → M08 会员等级页；积分余额 → M06 积分总览页；兑换记录 → M14；心愿单 → M11；我的券包 → M16；个人信息编辑 → M04；亲子绑定管理 → M05；消息中心 → M18 |
| **M04 个人信息编辑页** | M03 个人信息编辑 | 保存成功 → 返回 M03 个人中心 |
| **M05 亲子绑定页** | M03 亲子绑定管理 | 返回 → M03 个人中心（具体返回方式待实现确认） |
| **M06 积分总览页** | M03 点击积分；M00 点击积分入口 | 大RF「查看明细」→ M07（大RF）；小RF「查看明细」→ M07（小RF）；「去商城兑换」→ M09 积分商城页 |
| **M07 积分明细页** | M06 查看明细 | 返回 → M06 积分总览（无其他跳转） |
| **M08 会员等级页** | M03 会员卡片 / 等级说明；M00 头像区会员标识 | 返回 → M03 个人中心（无其他跳转） |
| **M09 积分商城页** | M00 点击大RF/小RF介绍卡片进入；M06「去商城兑换」 | 点击商品卡片 → M10；心愿单入口 → M11；购物车图标 → M12 |
| **M10 商品详情页** | M09 点击商品；M11 心愿单点击商品 | 加入心愿单/购物车 → Toast（加购后可从 M09 购物车图标进 M12）；立即兑换 → M13 兑换确认页 |
| **M11 心愿单页** | M03 心愿单；商城页心愿单入口 | 点击商品 → M10 商品详情页；移入购物车/取消收藏（本页操作） |
| **M12 购物车页** | M09 购物车图标；M03 入口（若个人中心提供购物车入口，见分析） | 「去兑换」→ M13 兑换确认页 |
| **M13 兑换确认页** | M10「立即兑换」；M12「去兑换」 | 确认兑换成功 → M15 核销码页 |
| **M14 兑换记录页** | M03 兑换记录 | 「查看核销码」→ M15 核销码页；「查看详情」→ 订单详情（若单独有页则补充） |
| **M15 核销码页** | M13 兑换成功后自动跳转；M14「查看核销码」 | 返回 → 上一页（M13 或 M14） |
| **M16 我的券包页** | 底部 Tab「我的券包」；M03 我的券包 | 点击未使用/已使用/已过期券 → M17 券详情页 |
| **M17 券详情页** | M16 点击某张券 | 返回 → M16 我的券包 |
| **M18 消息中心页** | M03 消息中心 | 无二次跳转（仅列表浏览） |

## 二、底部 Tab 与页面对应

## 三、关键流程串联

- **登录流程**：M01 →（已绑定）M00 或（未绑定）M02 → M03 或 M00  
- **兑换流程**：M09 或 M00 → M10 → M13 → M15；或 M12 → M13 → M15  
- **积分查看**：M00 / M03 → M06 → M07  
- **券使用**：M16 → M17（核销码展示）
