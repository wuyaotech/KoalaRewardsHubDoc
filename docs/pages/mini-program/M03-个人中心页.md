# M03 个人中心页（会员中心）

**所属模块**：会员信息 / 综合入口

**触发场景**：底部 Tab "会员中心"（第三个 Tab）

**路径**：`pages/home/home`

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 状态栏占位 | 顶部安全区 |
| 用户信息 | 显示"Hi,姓名"和手机号（中间四位脱敏），未登录显示"Hi,会员" |
| 会员等级卡片 | 横向轮播（Swiper），展示所有等级卡片，当前等级居中 |
| 子女积分卡片 | 每个绑定的小朋友一张卡片，展示小RF积分余额 + "去兑换"入口 |
| 亲子关系入口 | "+ 亲子关系"按钮，跳转亲子绑定管理 |
| 数据概览 | 三项并排：RF积分 / 待核销 / 好礼券 |
| 菜单列表 | 兑换记录 / 购物车 / 我的消息 / 账号管理 / 联系客服 / 条款与协议 |
| 底部 TabBar | 自定义 TabBar，当前选中"会员中心" |

### 会员等级卡片

- 横向 Swiper，前后留白（previous-margin / next-margin），当前等级卡片居中高亮
- 每张卡片：等级编号（V1/V2/...）、等级名称、标签（"当前等级"/"待解锁"）、进度描述
- 当前等级显示"累计积分 X/Y"；已超越的等级显示"已超越该等级"；待解锁显示"还需N积分晋级"
- 满级时显示"已满级"

### 数据概览

| 项目 | 点击跳转 |
|------|---------|
| RF积分 | 无跳转 |
| 待核销 | 跳转兑换记录页（pending tab） |
| 好礼券 | 跳转优惠券列表页 |

### 菜单列表

| 菜单项 | 跳转目标 |
|--------|----------|
| 兑换记录 | `/pages/redeem-records/redeem-records` |
| 购物车 | `/pages/cart/cart` |
| 我的消息 | `/pages/message/message` |
| 账号管理 | `/pages/account-manage/account-manage`（M04） |
| 联系客服 | Toast "敬请期待"（暂未实现） |
| 条款与协议 | `/pages/agreement-list/agreement-list` |

## 交互说明

- 未登录 → 自动弹出登录弹窗（login-dialog）
- 登录成功 → 刷新用户信息、积分汇总、等级卡片
- 每次 onShow 都会重新加载数据（确保从其他页面返回时数据最新）
- "+ 亲子关系" → 跳转 `/pages/children-manage/children-manage`
- 子女积分卡片"去兑换" → 跳转积分商城

## 关联接口

| 接口 | 用途 |
|------|------|
| `GET /members/me` | 获取当前用户信息（姓名、手机号等） |
| `GET /points/summary` | 获取积分汇总（大RF余额、子女小RF余额） |
| `POST /orders/list` | 获取订单列表，计算待核销数量 |
| `GET /member-levels/all` | 获取所有等级列表 |
| `GET /member-levels/me` | 获取当前用户等级信息 |
