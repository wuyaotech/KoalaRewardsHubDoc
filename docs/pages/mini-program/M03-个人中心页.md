# M03 个人中心页

**所属模块**：会员信息 / 综合入口

**触发场景**：底部 Tab "我的"

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 会员卡片 | 头像、昵称、会员等级图标、等级名称 |
| 积分概览 | 大RF积分余额 / 小RF积分余额（如有绑定小朋友） |
| 快捷入口 | 兑换记录、心愿单、我的券包 |
| 功能列表 | 个人信息编辑、亲子绑定管理、会员等级说明、消息中心 |

## 交互说明

- 点击会员卡片 → 跳转 M08 会员等级页
- 点击积分余额 → 跳转 M06 积分总览页
- 点击各快捷入口 → 跳转对应页面
- 未绑定小朋友时不展示小RF积分区域
