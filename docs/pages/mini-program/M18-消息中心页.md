# M18 消息中心页

**所属模块**：消息通知

**触发场景**：个人中心 → 消息中心（有未读时显示红点）

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 消息列表 | 时间倒序的系统消息列表 |
| 未读标记 | 未读消息左侧有蓝点标识，进入页面后自动全部标为已读 |
| 空状态 | 无消息时展示"暂无消息" |

**消息列表每条记录包含**：

| 字段 | 说明 |
|------|------|
| 消息内容 | 如"您的积分将于XX月XX日清零，请尽快使用" |
| 消息时间 | 触发时间 |
| 已读/未读 | 状态标记 |

**消息类型**：

| 类型 | 示例内容 |
|------|---------|
| 积分即将清零 | 您的积分将于XX月XX日清零，请尽快使用 |
| 兑换成功 | 您已成功兑换「商品名称」，请前往核销 |
| 核销完成 | 您兑换的「商品名称」已核销完成 |

## 交互说明

- 进入页面后所有未读消息自动标为已读，红点消失
- 支持下拉加载更多
