# M16 我的券包页

**所属模块**：优惠券

**触发场景**：个人中心 → 我的券包

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 状态 Tab | 未使用 / 已使用 / 已过期 |
| 券列表 | 每张券的卡片式展示 |

**每张券卡片包含**：

| 字段 | 说明 |
|------|------|
| 券面额/折扣 | 如 "满100减20" 或 "8折券" |
| 券名称 | 券的标题 |
| 有效期 | 起止日期 |
| 使用规则 | 简要使用条件 |
| 状态标记 | 未使用 / 已使用 / 已过期 |

## 交互说明

- 点击未使用的券 → 跳转 M17 券详情页（含核销码）
- 已使用/已过期的券 → 点击可查看详情但核销码失效
- 空状态 → 引导去商城用积分兑换
