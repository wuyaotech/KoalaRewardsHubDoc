# M00 H5 首页

**所属模块**：会员信息 + 积分账户（综合入口）

**触发场景**：微信授权、手机号绑定完成后进入的首页；底部 Tab 为「积分商城 / 会员中心 / 我的券包」时的主框架首屏。

**设计参考**：客户原始需求中的首页草图（`docs/客户原始需求/0af2f4a0-80c1-4f99-9e75-7c837380d82d.png`）

---

## 布局要点

### 1. 头像区（首屏顶部）

| 位置 | 内容 |
|------|------|
| **左侧** | 用户头像（圆形）+ 称呼「XX女士/先生」；下方展示**当前可用积分**（可展示大RF或总积分入口）；再下为**会员等级**标识（如「考拉会员」） |
| **右侧** | 可放置品牌形象/考拉插画，增强辨识度 |

- 头像、称呼来自会员信息；积分来自积分账户；等级来自会员等级模块。
- 点击头像或会员标识可进入个人中心/会员等级页；点击积分可进入积分总览或积分明细。

### 2. 两种积分介绍（首屏中部）

在首屏头像区下方，用两块区域分别介绍两种积分，便于用户理解并引导进入对应商城或规则页：

| 积分类型 | 说明文案要点 | 交互 |
|----------|----------------|------|
| **大RF积分** | 消费积分：线下消费由工作人员录入（1元=1积分），用于兑换大RF商品，与会员等级挂钩 | 可点击进入「大RF积分兑换规则」或「大RF商城」 |
| **小RF积分** | 上课积分：给小朋友的，根据上课表现由工作人员录入，用于兑换小RF商品，与等级无关 | 可点击进入「小RF积分兑换规则」或「小RF商城」；未绑定小朋友时可仅展示说明或弱化入口 |

- 未绑定小朋友时，小RF积分区域可仅做简要说明或折叠，不强调入口。

### 3. 底部导航（Tab）

| Tab | 跳转 |
|-----|------|
| 积分商城 | M09 积分商城页 |
| 会员中心 | M03 个人中心页 |
| 我的券包 | M16 我的券包页 |

---

## 与现有页面的关系

- **M03 个人中心**：头像区信息与个人中心会员卡片一致，首页做精简展示。
- **M06 积分总览**：首页仅展示积分入口或汇总数字，详情进 M06。
- **M09 积分商城**：作为 Tab 之一，可与首页首屏「两种积分介绍」联动，点击后切到对应大RF/小RF商城。

---

## 小结

- **头像**：圆形头像 + 称呼 + 积分 + 会员等级，右侧可配考拉形象。
- **两种积分**：大RF（消费积分）、小RF（给小朋友的上课积分），首屏简要介绍并引导至规则或商城。
