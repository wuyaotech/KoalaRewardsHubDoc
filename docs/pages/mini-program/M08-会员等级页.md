# M08 会员等级页

**所属模块**：会员等级

**触发场景**：个人中心 → 会员卡片 / 等级说明入口

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 当前等级 | 等级图标 + 等级名称（如 "小熊会员"） |
| 等级进度 | 进度条展示距下一等级所需累计积分差距 |
| 等级列表 | 所有等级（考拉→小熊→小马→海豚→花豹）的积分门槛和权益说明 |
| 当前等级权益 | 高亮展示当前等级已享有的权益 |

## 交互说明

- 仅与大RF积分关联，页面上标明 "根据大RF累计获得积分计算等级"
- 积分清零后等级不降级，仅影响当前可用积分
- 等级权益内容由后台配置，前端动态渲染
