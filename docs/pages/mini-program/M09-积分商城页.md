# M09 积分商城页

**所属模块**：商品管理 / 积分账户

**路径**：`pages/mall/mall`

**触发场景**：底部 Tab "积分商城"（第二个 Tab）

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 顶部导航 | 自定义导航栏，左侧 Tab 切换"RF超市" / "rf幼崽超市" |
| 积分余额卡片 | 积分图标 + 标签 + 当前余额 + "明细 >" 入口 + 副文本提示 |
| 筛选栏 | 商品分类下拉 + 积分排序（升序/降序切换） |
| 商品列表 | 两列瀑布式布局，product-card 组件 |
| 浮动按钮 | 滚动后出现：购物车 + 返回顶部 |
| 底部 TabBar | 自定义 TabBar，当前选中"积分商城" |

### 积分类型切换

- 顶部导航内嵌 Tab：**RF超市**（大RF）和 **rf幼崽超市**（小RF）
- 未绑定小朋友的用户隐藏"rf幼崽超市"Tab
- 切换 Tab 时重新加载积分汇总、分类列表和商品列表

### 积分余额卡片

| 状态 | 余额显示 | 副文本 |
|------|---------|--------|
| 未登录 | `--` | "登录后可查看当前积分余额" |
| 大RF（正常） | 实际余额 | 无 |
| 大RF（临期） | 实际余额 | "您有N积分将于YYYY-MM-DD清零" |
| 小RF | 所有小朋友积分合计 | "已汇总 N 个小朋友积分账户" |
| 获取失败 | `--` | "积分信息暂时获取失败" |

### 筛选与排序

- **分类筛选**：下拉菜单，包含"全部" + 后端返回的分类列表，选择后重新请求商品
- **积分排序**：点击切换升序/降序（POINTS_ASC / POINTS_DESC），箭头方向跟随

### 商品列表

- 每个商品卡片展示：封面图、名称、积分价格、售罄标记
- 支持心愿单（收藏）功能，未登录点收藏跳登录页
- 点击商品 → 跳转 `/pages/product-detail/product-detail?uuid=xxx`

### 浮动按钮

- 页面滚动超过 150px 后右下角浮出
- 购物车按钮 → 跳转 `/pages/cart/cart`
- 返回顶部按钮 → 平滑滚动回顶部
- 浮动按钮出现时隐藏底部 TabBar，消失时恢复

## 页面参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| pointsType | `BIG_RF` \| `SMALL_RF` | `BIG_RF` | 初始积分类型 |

## 交互说明

- 大RF商城和小RF商城商品池完全隔离，切换后分类和商品都重新加载
- 下拉刷新 → 重新初始化整个页面（积分、分类、商品、心愿单）
- 滚动到底部 → 本地分页加载更多（前端一次拉取 100 条，每次显示 10 条）
- 分类下拉菜单展开时有蒙层遮罩

## 关联接口

| 接口 | 方法 | 用途 |
|------|------|------|
| `/points/summary` | GET | 获取积分汇总（大RF余额、小RF分子女余额、到期信息） |
| `/products/list` | POST | 商品列表（按 pointsType、分类、排序筛选） |
| `/product-categories/list` | POST | 商品分类列表（按 pointsType 筛选） |
| `/wishlist` | GET | 获取当前用户心愿单 |
| `/wishlist/add` | POST | 添加心愿单 |
| `/wishlist/remove` | POST | 移除心愿单 |

## 跳转关系

| 操作 | 目标 |
|------|------|
| 点击"明细 >" | `/pages/points-ledger/points-ledger`（M07） |
| 点击商品卡片 | `/pages/product-detail/product-detail?uuid=xxx` |
| 点击购物车浮动按钮 | `/pages/cart/cart` |
