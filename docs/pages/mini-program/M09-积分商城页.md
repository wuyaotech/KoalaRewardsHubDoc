# M09 积分商城页

**所属模块**：商品管理

**触发场景**：底部 Tab "商城" / 积分总览页 "去商城兑换"

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 积分类型切换 | 顶部 Tab：大RF商城 / 小RF商城 |
| 当前积分余额 | 展示当前所选类型的可用积分 |
| 分类筛选 | 商品分类横向滚动标签 |
| 商品列表 | 卡片式布局：商品图片、名称、所需积分 |
| 排序 | 按积分从低到高 / 从高到低 / 最新上架 |

## 交互说明

- 大RF商城和小RF商城商品池完全隔离
- 未绑定小朋友的用户不展示小RF商城 Tab
- 点击商品卡片 → 跳转 M10 商品详情页
- 支持下拉加载更多
