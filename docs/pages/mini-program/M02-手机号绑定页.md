# M02 手机号绑定页

**所属模块**：用户与认证

**触发场景**：微信授权登录后，首次使用需绑定手机号，并尝试匹配已有会员档案

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 提示文案 | "绑定手机号，开启积分之旅" |
| 快捷获取 | "使用微信绑定的手机号" 按钮（微信 getPhoneNumber） |
| 手动输入 | 手机号输入框 + 短信验证码（备选方案） |
| 提交按钮 | 确认绑定 |

## 交互说明

- 若系统中存在同手机号的线下会员档案 → 自动绑定该档案，并将会员状态更新为"已激活"
- 若系统中不存在同手机号会员档案 → 仅完成小程序注册，用户进入"已注册未入会"状态
- 绑定成功 → 跳转个人中心（M03）或首页
- 手机号已被其他账号绑定 → 提示冲突，引导联系工作人员
