# M04 个人信息编辑页（账号管理）

**所属模块**：会员信息

**触发场景**：会员中心 → 账号管理

**路径**：`pages/account-manage/account-manage`

---

## 页面结构

| 区域 | 内容 |
|------|------|
| 导航栏 | 自定义导航栏，标题"账号管理" |
| 信息卡片 | 姓名 / 手机号 / 生日 / 性别 / 亲子关系，每行一个字段 |
| 退出登录 | 底部红色按钮 |

## 字段说明

| 字段 | 展示 | 编辑方式 | 说明 |
|------|------|----------|------|
| 姓名 | "姓名"（拼接后展示） | 点击 → 弹窗输入姓和名 | 未设置显示"未设置" |
| 手机号 | 脱敏展示（138\*\*\*\*1234） | 只读，不可修改 | 未绑定显示"未绑定" |
| 生日 | 日期格式 YYYY-MM-DD | 点击 → 日期选择器 | 选择后立即保存 |
| 性别 | 男 / 女 | 点击 → 弹窗选择 | 选择后立即保存 |
| 亲子关系 | 子女姓名逗号分隔 | 点击 → 跳转亲子管理页 | 未设置显示"未设置" |

**编辑特点**：每个字段独立保存（修改即提交），没有统一的"保存"按钮。

## 弹窗：修改姓名

| 字段 | 说明 |
|------|------|
| 姓 | 必填 |
| 名 | 必填 |
| 按钮 | 「取消」/「确定」 |

## 弹窗：选择性别

- 展示"男"/"女"两个选项，当前选中项高亮
- 选择后立即保存并关闭弹窗

## 退出登录

- 点击 → 二次确认弹窗"确定要退出登录吗？"
- 确认 → 清除 token 和用户信息 → 跳转首页（switchTab index）

## 交互说明

- 未登录 → 跳转登录页
- 数据来源：调用 `/members/me` 获取 account + member 信息，优先展示 member 上的字段
- 修改姓名/生日/性别 → 调用 `/members/update-profile` 单字段更新
- 修改成功 → Toast "修改成功"

## 关联接口

| 接口 | 用途 |
|------|------|
| `GET /members/me` | 加载账号和会员档案信息 |
| `POST /members/update-profile` | 更新姓名 / 生日 / 性别 |
