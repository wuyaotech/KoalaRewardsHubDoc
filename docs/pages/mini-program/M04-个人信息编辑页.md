# M04 个人信息编辑页

**所属模块**：会员信息

**触发场景**：个人中心 → 个人信息编辑

---

## 主要内容

| 字段 | 类型 | 说明 |
|------|------|------|
| 头像 | 图片上传 | 微信头像或自定义上传 |
| 姓名 | 文本输入 | 真实姓名 |
| 手机号 | 只读展示 | 已绑定手机号，不可直接修改 |
| 生日 | 日期选择 | 年月日 |
| 性别 | 单选 | 男 / 女 |
| 保存按钮 | — | 提交修改 |

## 交互说明

- 手机号如需更换，需联系工作人员在后台操作
- 保存成功 → Toast 提示 + 返回个人中心
