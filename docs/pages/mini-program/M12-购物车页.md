# M12 购物车页

**所属模块**：兑换交易

**触发场景**：商城页购物车图标 / 个人中心入口

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 商品列表 | 已加入购物车的商品（图片、名称、所需积分、数量） |
| 勾选操作 | 勾选要兑换的商品（可多选） |
| 积分小计 | 底部实时显示已勾选商品的积分总计 |
| 当前余额 | 展示对应积分类型的可用余额 |
| 底部操作栏 | "去兑换" 按钮 |
| 空状态 | 无商品时引导去商城 |

## 交互说明

- 大RF商品和小RF商品在购物车中分组展示
- 积分不足时 "去兑换" 按钮提示积分不足
- 支持修改数量、移除商品
- 点击 "去兑换" → 跳转 M13 兑换确认页
