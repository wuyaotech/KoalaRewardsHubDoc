# M12 购物车页（含心愿单）

**所属模块**：兑换交易

**路径**：`pages/cart/cart`

**触发场景**：商城页购物车浮动按钮 / 商品详情页购物车图标 / 首页快捷入口 / 会员中心菜单

---

## 页面结构

页面顶部导航栏内嵌两个 Tab：**购物车** 和 **心愿单**，共用同一页面。

### 一、购物车 Tab

| 区域 | 内容 |
|------|------|
| 筛选子 Tab | 全部 / RF超市 / rf幼崽超市 / 失效 |
| 商品列表 | 勾选框 + 商品图片 + 名称 + 规格文案 + 所需积分 + 数量加减 |
| 失效商品 | 下架或库存为0的商品，勾选框置灰，显示失效原因，提供删除按钮 |
| 底部操作栏 | 全选按钮 + 大RF积分小计 + 小RF积分小计 + 已选件数 + "立即兑换"按钮 |
| 空状态 | "您的购物车中没有可选商品，快去选购吧！" + "去选购"跳转商城 |

### 二、心愿单 Tab

| 区域 | 内容 |
|------|------|
| 商品列表 | product-card 组件网格展示已收藏商品 |
| 空状态 | "心愿单还是空的，去逛逛积分商城吧！" + "去选购"跳转商城 |

> 心愿单 Tab 无底部操作栏，点击商品卡片跳转商品详情页。

## 交互说明

### 购物车

- **筛选子 Tab**：全部展示所有有效+失效商品；RF超市/rf幼崽超市按积分类型筛选有效商品；失效 tab 只展示下架或无库存的商品
- **勾选**：点击商品左侧勾选框选中/取消，失效商品不可勾选
- **全选**：底部全选按钮控制当前筛选条件下所有有效商品的勾选状态
- **数量修改**：点击 +/− 调用 `POST /cart/update` 修改数量，最小为1，最大为当前 SKU 库存
- **删除**：失效商品右侧显示删除按钮，调用 `POST /cart/remove`
- **立即兑换**：校验积分余额是否充足 → 弹出 checkout-sheet（结算明细弹窗）
- **未登录**：进入页面时检测登录态，未登录跳转登录页

### 心愿单

- 点击商品 → 跳转 `/pages/product-detail/product-detail?uuid=xxx`
- 心愿单数据来源 `GET /wishlist`

### checkout-sheet（结算明细弹窗）

- 展示已选商品列表（图片、名称、规格、积分、数量）
- 大RF和小RF分别显示积分小计，余额不足时标红提示
- 包含小RF商品时需选择扣减积分的小朋友（从积分汇总接口获取子女列表）
- 未选择小朋友或余额不足时"确认兑换"按钮置灰
- 确认兑换 → 调用 `POST /orders/create` 创建订单

## 关联接口

| 接口 | 方法 | 用途 |
|------|------|------|
| `/cart` | GET | 获取购物车列表 |
| `/cart/update` | POST | 修改购物车商品数量 |
| `/cart/remove` | POST | 移除购物车商品 |
| `/wishlist` | GET | 获取心愿单列表 |
| `/points/summary` | GET | 获取积分余额和子女信息 |
| `/orders/create` | POST | 创建兑换订单 |

## 跳转关系

| 操作 | 目标 |
|------|------|
| 点击心愿单商品 | `/pages/product-detail/product-detail?uuid=xxx` |
| 点击"去选购" | `switchTab` 到商城页 |
