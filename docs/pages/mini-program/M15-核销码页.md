# M15 核销码页

**所属模块**：核销

**触发场景**：兑换成功后自动跳转 / 兑换记录 → 查看核销码

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 商品信息 | 兑换的商品名称、数量 |
| 核销二维码 | 大尺寸二维码，供前台扫码 |
| 核销数字码 | 二维码下方显示数字码（备用，前台可手动输入） |
| 核销状态 | 待核销 / 已核销 |
| 核销时间 | 已核销时显示核销完成时间 |
| 有效期说明 | 核销码自兑换成功起7天内有效 |
| 使用说明 | "请到前台出示此码完成兑换" |

## 交互说明

- 待核销状态：二维码正常展示，可被扫码
- 已核销状态：二维码置灰，显示 "已兑换" 水印
- 超过7天未核销：显示"已过期"，二维码失效
- 核销完成后页面自动刷新状态（或用户手动刷新）
