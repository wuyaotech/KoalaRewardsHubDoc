# M10 商品详情页

**所属模块**：商品管理

**路径**：`pages/product-detail/product-detail`

**触发场景**：商城列表 → 点击商品卡片 / 购物车心愿单 Tab → 点击商品

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 商品图片 | 轮播大图（`product.images`），带圆点指示器 |
| 商品名称 | 商品标题 |
| 限购提示 | 如"每月限兑 X 次"（有限购规则时展示） |
| 所需积分 | 商品统一兑换积分，带积分类型图标 |
| 商品详情 | 装修块（`descriptionBlocks` 文字块/图片块）优先渲染；降级为 `description` 富文本；再降级为详情图片；均无时显示"暂无商品详情" |
| 底部操作栏 | 购物车图标（带徽标） + 心愿单图标 + "加购" + "立即兑换" |

## 交互说明

### 底部操作栏

- **购物车图标**：显示当前购物车商品数量徽标，点击跳转购物车页
- **心愿单图标**：心形图标，已收藏时实心/未收藏时空心，点击切换收藏状态（调用 `/wishlist/add` 或 `/wishlist/remove`）
- **加购**：有规格时弹出 spec-sheet 选择规格后加入购物车；无规格时直接加入（调用 `POST /cart/add`）
- **立即兑换**：有规格时弹出 spec-sheet 选择规格；无规格时直接进入兑换流程 → 弹出 checkout-sheet 确认 → 创建订单
- **已售罄**：商品所有 SKU 库存为 0 时，"加购"和"立即兑换"替换为"已售罄"置灰按钮

### spec-sheet（规格选择弹窗）

- 展示商品图片、名称、积分
- 按规格组列出规格值标签，点击切换选中
- 选中主规格组的规格值时，若该值配有规格图片则更新弹窗内预览图
- 根据选中的规格组合匹配 SKU，显示对应库存状态
- SKU 无库存时"确定"按钮显示为"已售罄"置灰

### checkout-sheet（结算明细弹窗）

- 展示待兑换商品信息（图片、名称、规格、积分、数量）
- 大RF/小RF 分别显示积分小计和余额对比，不足时标红
- 包含小RF商品时需选择扣减积分的小朋友
- 确认兑换 → 先加入购物车 → 调用 `POST /orders/create` 创建订单

### 兑换流程

```
点击"立即兑换"
  ├─ 有规格 → spec-sheet 选规格 → onSpecSheetConfirm
  └─ 无规格 → 直接 doRedeem
        ↓
  校验积分余额
        ↓
  弹出 checkout-sheet
        ↓
  确认 → POST /cart/add → POST /orders/create
```

## 页面参数

| 参数 | 类型 | 说明 |
|------|------|------|
| uuid | string | 商品 UUID |

## 关联接口

| 接口 | 方法 | 用途 |
|------|------|------|
| `/products/:uuid/detail` | GET | 商品详情 |
| `/points/summary` | GET | 积分余额和子女信息 |
| `/wishlist` | GET | 判断是否已收藏 |
| `/wishlist/add` | POST | 添加收藏 |
| `/wishlist/remove` | POST | 取消收藏 |
| `/cart/add` | POST | 加入购物车 |
| `/cart` | GET | 获取购物车数量（徽标） |
| `/orders/create` | POST | 创建兑换订单 |
