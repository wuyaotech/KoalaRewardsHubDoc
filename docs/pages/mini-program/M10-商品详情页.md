# M10 商品详情页

**所属模块**：商品管理

**触发场景**：商城列表 → 点击商品卡片

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 商品图片 | 轮播大图 |
| 商品名称 | 商品标题 |
| 所需积分 | 兑换该商品需要的积分数 |
| 库存信息 | 剩余可兑换数量 |
| 限购规则 | 如 "每人每月限兑1件"（如有） |
| 商品参数 | 规格、描述等详细信息 |
| 底部操作栏 | "加入心愿单"、"加入购物车"、"立即兑换" |

## 交互说明

- 积分不足时 "立即兑换" 和 "加入购物车" 置灰，提示积分不足
- 库存为0时显示 "已兑完"
- 超过限购数量时提示 "已达兑换上限"
- "加入心愿单" → 收藏成功 Toast
- "加入购物车" → 加入成功 Toast
- "立即兑换" → 跳转 M13 兑换确认页
