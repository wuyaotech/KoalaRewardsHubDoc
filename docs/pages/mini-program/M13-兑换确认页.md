# M13 兑换确认页

**所属模块**：兑换交易

**触发场景**：商品详情 "立即兑换" / 购物车 "去兑换"

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 商品信息 | 兑换的商品列表（图片、名称、数量、单项积分） |
| 积分汇总 | 合计所需积分 |
| 当前余额 | 兑换前的可用积分 |
| 兑换后余额 | 兑换后预估剩余积分 |
| 确认按钮 | "确认兑换" |

## 交互说明

- 确认兑换 → 弹出二次确认弹窗 "确认花费 XXX 积分兑换？"
- 确认后扣减积分，生成订单，跳转 M15 核销码页
- 兑换失败（库存不足/积分变动等） → 提示原因
