# M07 积分明细页

**所属模块**：积分账户

**路径**：`pages/points-ledger/points-ledger`

**触发场景**：积分商城页积分卡片 → "明细 >"

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 导航栏 | 自定义导航栏（custom-nav-bar），标题"RF积分明细" |
| 筛选标签 | 全部 / 收入 / 支出 / 过期清零，胶囊式 Tag 切换 |
| 流水列表 | 按月分组（sticky 月份标题），时间倒序 |

### 流水列表每条记录

| 字段 | 说明 |
|------|------|
| 变动描述 | reason 字段，缺省时按类型显示（"积分录入"/"积分过期"/"积分支出"） |
| 时间 | `MM月DD日 HH:mm:ss` 格式 |
| 积分数 | 收入显示 `+N`（橙色），支出/过期显示 `-N`（灰色） |

### 月份分组

- 按 `YYYY年M月` 分组，月份标题 sticky 置顶
- 同一月份内按时间倒序排列

## 页面参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| pointsType | `BIG_RF` \| `SMALL_RF` | `BIG_RF` | 积分类型 |

## 交互说明

- 切换筛选标签 → 重置列表，重新请求
- 滚动到底部 → 自动加载下一页（每页 50 条）
- 空状态显示"暂无积分记录"

## 关联接口

| 接口 | 方法 | 用途 |
|------|------|------|
| `/points/ledger/list` | POST | 分页查询积分流水，支持按 pointsType、ledgerType 筛选 |

### 请求参数

```json
{
  "pointsType": "BIG_RF",
  "page": 1,
  "limit": 50,
  "ledgerType": "IN"
}
```

### 返回结构

```json
{
  "success": true,
  "data": {
    "list": [
      {
        "uuid": "xxx",
        "type": "IN",
        "amount": "100",
        "balanceAfter": "1500",
        "reason": "消费录入",
        "relatedType": null,
        "relatedUuid": null,
        "createdAt": "2026-03-15T10:30:00.000Z"
      }
    ],
    "pagination": { "page": 1, "limit": 50, "total": 120 }
  }
}
```
