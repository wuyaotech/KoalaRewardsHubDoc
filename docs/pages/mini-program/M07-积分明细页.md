# M07 积分明细页

**所属模块**：积分账户

**触发场景**：积分总览页 → 查看明细

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 积分类型标识 | 页面顶部标明当前查看的是大RF积分还是小RF积分 |
| 当前余额 | 大字展示当前可用余额 |
| 流水列表 | 时间倒序的积分变动记录 |

**流水列表每条记录包含**：

| 字段 | 说明 |
|------|------|
| 变动类型 | 收入 / 支出 / 过期清零 |
| 变动描述 | 如 "消费录入"、"兑换商品XXX"、"积分过期" |
| 积分数 | +100 / -50 等 |
| 时间 | 变动时间 |

## 交互说明

- 支持下拉加载更多（分页）
- 可按 "全部 / 收入 / 支出 / 过期" 筛选
