# M06 积分总览页

**所属模块**：积分账户

**触发场景**：个人中心点击积分区域 / 底部 Tab（如有）

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 大RF积分卡片 | 当前可用积分余额、"查看明细" 入口 |
| 小RF积分卡片 | 绑定的小朋友姓名 + 积分余额、"查看明细" 入口（未绑定不展示） |
| 快捷入口 | "去商城兑换" 按钮 |

## 交互说明

- 点击大RF积分 "查看明细" → 跳转 M07 积分明细页（大RF类型）
- 点击小RF积分 "查看明细" → 跳转 M07 积分明细页（小RF类型）
- 点击 "去商城兑换" → 跳转 M09 积分商城页
