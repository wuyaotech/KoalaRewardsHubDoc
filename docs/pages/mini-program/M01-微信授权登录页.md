# M01 微信授权登录页

**所属模块**：用户与认证

**触发场景**：用户首次打开小程序 / 登录态过期 / 未登录时访问需登录的功能

---

## 实现方式

登录流程通过 `login-dialog` 底部弹窗组件实现（非独立页面），分三步完成。
该组件在首页、会员中心等需要登录的页面中按需弹出；也可通过 `/pages/login/login` 页面作为入口调用。

---

## Step 1：登录方式选择

| 区域 | 内容 |
|------|------|
| Logo | RALLY FIELD TENNIS CLUB 品牌 Logo |
| 标题 | "欢迎来到RALLY FIELD TENNIS CLUB" |
| 副标题 | "登录解锁积分权益" |
| 主按钮 | "微信一键登录"（微信 getPhoneNumber 授权，同时获取手机号） |
| 次按钮 | "其他手机号登录" → 进入 Step 2 |
| 隐私协议 | 勾选"我已阅读并同意《用户协议》和《隐私政策》"，未勾选不可登录 |

**交互说明**：
- 点击"微信一键登录" → 微信授权弹窗，同时获取 openid 和手机号
  - 授权成功 + 已有姓名信息 → 登录成功，关闭弹窗
  - 授权成功 + 新用户（无姓名）→ 进入 Step 3 注册表单
  - 授权拒绝 → 提示"需要授权手机号才能登录"
- 点击"其他手机号登录" → 进入 Step 2（需先勾选协议）
- 点击《用户协议》/《隐私政策》→ 跳转到协议详情页

## Step 2：手机号验证码登录

| 区域 | 内容 |
|------|------|
| Logo + 标题 | 同 Step 1 |
| 手机号输入 | 11 位手机号，数字键盘 |
| 验证码输入 | 6 位数字，右侧「获取验证码」按钮 |
| 错误提示 | 红色文字展示在表单下方 |
| 隐私协议 | 同 Step 1 |
| 底部按钮 | 「取消」返回 Step 1 / 「登录」提交 |

**验证码规则**：
- 点击获取后 60 秒倒计时
- 更换手机号后倒计时和已输入验证码自动清空
- 倒计时状态在组件重新打开时自动恢复（存 Storage）
- 开发环境下验证码会自动回填

**交互说明**：
- 登录成功 + 已有姓名 → 关闭弹窗
- 登录成功 + 无姓名 → 进入 Step 3
- 验证码错误 → 提示"验证码错误或已过期"
- 手机号格式错误 → 提示"请输入正确的手机号"

## Step 3：新用户注册信息填写

| 区域 | 内容 |
|------|------|
| 标题 | "欢迎来到RALLY FIELD TENNIS CLUB" |
| 副标题 | "请完善个人信息完成会员注册" |
| 右上角 | 「清空」按钮 +「✕」关闭按钮 |
| 姓氏 | 必填 |
| 名字 | 必填 |
| 生日 | 选填，日期选择器 |
| 性别 | 选填，男 / 女 |
| 提交按钮 | "快速注册" |
| 跳过链接 | "暂时跳过" |

**交互说明**：
- 提交注册 → 调用 `/members/update-profile` 更新个人信息 → 登录成功
- 点击"暂时跳过" → 直接登录成功（不填写信息）
- 姓氏或名字为空 → Toast 提示

## 关联接口

| 接口 | 用途 |
|------|------|
| `POST /auth/wechat-login` | 微信一键登录（code + phoneCode） |
| `POST /auth/send-code` | 发送短信验证码 |
| `POST /auth/phone-login` | 手机号 + 验证码登录 |
| `POST /members/update-profile` | 提交注册信息（姓名/生日/性别） |
