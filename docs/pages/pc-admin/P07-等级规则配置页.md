# P07 等级规则配置页

**所属模块**：会员等级

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 等级列表 | 所有等级的配置表格 |
| 编辑操作 | 每个等级可编辑 |

**每个等级的配置项**：

| 字段 | 说明 |
|------|------|
| 等级名称 | 如 考拉、小熊、小马、海豚、花豹 |
| 等级图标 | 上传等级对应的图标 |
| 积分门槛 | 升至该等级所需的最低大RF积分 |
| 权益内容 | 富文本描述该等级权益 |
| 排序 | 等级顺序 |

## 交互说明

- 仅关联大RF积分（消费积分）
- 修改门槛后，系统自动重新计算所有会员等级（需确认是否实时生效还是定时任务）
- 操作留痕，记入操作日志
