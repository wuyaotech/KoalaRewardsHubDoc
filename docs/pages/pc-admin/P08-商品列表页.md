# P08 商品列表页

**所属模块**：商品管理

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 筛选条件 | 积分类型（大RF/小RF）、商品分类、商品状态（上架/下架/仓库） |
| 搜索 | 按商品名称搜索 |
| 操作按钮 | "新建商品" |
| 商品表格 | 商品列表 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 商品图片 | 缩略图 |
| 商品名称 | 标题 |
| 积分类型 | 大RF / 小RF |
| 所需积分 | 商品统一兑换积分 |
| 库存 | 剩余库存 / 总库存 |
| 状态 | 草稿 / 仓库 / 已上架 / 已下架 |
| 兑换量 | 累计被兑换次数 |
| 操作 | 编辑、上架/下架、删除 |

## 交互说明

- "新建商品" → 跳转 P09 商品编辑页（新建模式）
- "编辑" → 跳转 P09 商品编辑页（编辑模式）
- "上架/下架" → 确认弹窗后切换状态
- 支持批量上架/下架（待确认）
