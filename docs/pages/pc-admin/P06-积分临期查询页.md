# P06 积分临期查询页

**所属模块**：积分账户

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 筛选条件 | 积分类型（大RF/小RF）、临期时间范围（如 1个月内/3个月内） |
| 临期会员表格 | 即将清零积分的会员列表 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 姓名 | 会员姓名 |
| 手机号 | 联系方式 |
| 积分类型 | 大RF / 小RF |
| 即将过期积分 | 将要清零的积分数 |
| 过期时间 | 清零日期 |
| 操作 | 查看详情 |

## 交互说明

- 便于工作人员人工提醒会员尽快使用积分
- 点击 "查看详情" → 跳转 P04 会员详情页
