# P10 商品分类管理页

**所属模块**：商品管理

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 分类列表 | 当前所有商品分类 |
| 操作按钮 | "新增分类" |

**分类列表字段**：

| 列 | 说明 |
|------|------|
| 分类名称 | 分类标题 |
| 排序 | 展示顺序 |
| 关联商品数 | 该分类下的商品数量 |
| 操作 | 编辑、删除 |

## 交互说明

- "新增分类" → 弹窗输入分类名称、排序
- "编辑" → 弹窗修改
- "删除" → 如分类下有商品则不允许删除，提示先移走商品
- 分类是否需要区分大RF/小RF → 待确认（可共用分类也可独立）
