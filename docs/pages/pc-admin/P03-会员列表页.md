# P03 会员列表页

**所属模块**：会员信息

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 搜索栏 | 按姓名、手机号搜索 |
| 筛选条件 | 会员等级、注册时间范围、是否绑定小朋友 |
| 会员表格 | 列表展示所有会员 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 姓名 | 会员姓名 |
| 手机号 | 绑定手机号 |
| 会员等级 | 当前等级名称 |
| 大RF积分 | 当前大RF可用积分 |
| 小RF积分 | 当前小RF可用积分（未绑定显示 "—"） |
| 注册时间 | 首次登录时间 |
| 操作 | 查看详情、录入积分 |

## 交互说明

- 点击 "查看详情" → 跳转 P04 会员详情页
- 点击 "录入积分" → 跳转 P05 积分录入页（预填该会员）
- 支持分页、排序
- 支持导出 Excel（待确认）
