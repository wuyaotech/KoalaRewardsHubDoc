# P03 会员列表页

**所属模块**：会员信息

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 搜索栏 | 按姓名、手机号搜索 |
| 筛选条件 | 会员等级、会员状态、建档来源、注册时间范围、是否绑定小朋友 |
| 操作区 | 支持 "新增会员/线下建档" |
| 会员表格 | 列表展示所有会员 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 姓名 | 会员姓名 |
| 手机号 | 绑定手机号 |
| 会员状态 | 未激活 / 已激活 / 已停用 |
| 建档来源 | 线下录入 / 后台补建 |
| 会员等级 | 当前等级名称 |
| 大RF积分 | 当前大RF可用积分 |
| 小RF积分 | 当前小RF可用积分（未绑定显示 "—"） |
| 注册时间 | 首次登录激活时间；未激活会员显示 "—" |
| 操作 | 查看详情、录入积分、编辑会员信息 |

## 交互说明

- 点击 "查看详情" → 跳转 P04 会员详情页
- 点击 "录入积分" → 跳转 P05 积分录入页（预填该会员）
- 点击 "新增会员/线下建档" → 弹窗或跳转表单页，录入手机号、姓名等基础信息后创建未激活会员档案
- 会员列表默认只展示正式会员档案；"已注册未入会用户"不进入本列表
- 支持分页、排序
- 支持导出 Excel（待确认）
