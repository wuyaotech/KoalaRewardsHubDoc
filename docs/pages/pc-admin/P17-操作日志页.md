# P17 操作日志页

**所属模块**：系统管理

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 筛选条件 | 操作类型、操作人、时间范围 |
| 日志表格 | 关键操作的审计记录 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 操作时间 | 精确到秒 |
| 操作人 | 工作人员姓名 |
| 操作类型 | 积分录入 / 积分更正 / 核销 / 商品上下架 / 账号管理 等 |
| 操作对象 | 涉及的会员/商品/订单 |
| 操作详情 | 具体变更内容（如 "会员张三 大RF积分 +500"） |
| IP地址 | 操作时的IP |

## 交互说明

- 日志只读，不可删除或修改
- 支持按时间范围导出
- 关键操作自动记录，无需手动触发
