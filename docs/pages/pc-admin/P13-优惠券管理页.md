# P13 优惠券管理页

**所属模块**：优惠券

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 操作按钮 | "创建券模板" |
| 券模板表格 | 所有优惠券模板列表 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 券名称 | 券的标题 |
| 券类型 | 代金券 / 折扣券 |
| 面额/折扣 | 如 "满100减20" / "8折" |
| 有效期 | 起止日期 |
| 使用规则 | 简要使用条件 |
| 已发放 | 已被兑换的数量 |
| 已核销 | 已使用核销的数量 |
| 状态 | 进行中 / 已过期 |
| 操作 | 编辑、查看核销记录 |

## 交互说明

- "创建券模板" → 弹窗或新页面填写券信息
- 券作为特殊商品在积分商城上架，此页面管理券模板本身
- "查看核销记录" → 弹窗展示该券的发放和核销明细
