# P01 登录页

**所属模块**：用户与认证

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 品牌标识 | 考拉积分管理后台 Logo + 系统名称 |
| 登录表单 | 手机号 + 密码 |
| 记住登录 | "记住我" 复选框 |
| 登录按钮 | 提交登录 |

## 交互说明

- 登录成功（管理员/超管）→ 跳转 P02 工作台首页
- 手机号/密码错误 → 提示错误信息
- 该手机号为普通用户（非管理员/超管）→ 提示 "无后台管理权限，请联系管理员"
- 该手机号未注册（未在小程序端注册过）→ 提示 "账号不存在，请先在小程序端注册"
