# P01 登录页

**所属模块**：用户与认证

---

## 页面结构

整体居中展示，白底卡片，上方为品牌标题，下方为登录表单。

| 区域 | 内容 |
|------|------|
| 品牌标识 | "考拉积分管理后台" + "RF商城 PC 管理后台" 副标题 |
| 登录方式切换 | Tab 切换：「密码登录」（默认）/ 「验证码登录」 |
| 登录按钮 | 提交登录 |

### Tab 1：密码登录

| 字段 | 说明 |
|------|------|
| 手机号 | 必填，11 位中国大陆手机号格式校验 |
| 密码 | 必填 |

### Tab 2：验证码登录

| 字段 | 说明 |
|------|------|
| 手机号 | 必填，11 位中国大陆手机号格式校验 |
| 验证码 | 6 位数字，输入框右侧内嵌「获取验证码」按钮 |

**验证码发送规则**：
- 点击「获取验证码」后进入 60 秒倒计时，倒计时期间按钮不可点击
- 如果服务端返回"请 N 秒后再试"，倒计时自动同步为 N 秒
- 开发环境下验证码会自动回填到输入框

## 交互说明

- 已登录状态（token 存在）→ 自动跳转 P02 工作台首页
- 密码登录成功（管理员/超管）→ 跳转 P02 工作台首页
- 验证码登录成功（管理员/超管）→ 跳转 P02 工作台首页
- 手机号/密码错误 → 表单上方红色 Alert 提示错误信息
- 验证码错误或过期 → 表单上方红色 Alert 提示
- 该手机号为普通用户（非管理员/超管）→ 提示"该账号无后台登录权限"
- 该手机号未注册 → 提示"该手机号未注册后台账号"
- 网络异常 → 提示"网络异常，请稍后重试"

## 关联接口

| 接口 | 用途 |
|------|------|
| `POST /auth/admin/login` | 密码登录 |
| `POST /auth/admin/phone-login` | 验证码登录 |
| `POST /auth/send-code` | 发送短信验证码 |
