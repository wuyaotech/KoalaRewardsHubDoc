# P04 会员详情页

**所属模块**：会员信息

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 基本信息 | 姓名、手机号、性别、生日、注册时间、微信昵称/头像 |
| 亲子信息 | 是否绑定小朋友、小朋友姓名列表 |
| 等级信息 | 当前等级、等级进度、历史等级变动记录 |
| 大RF积分 | 当前余额、累计获得、累计消耗、最近流水记录 |
| 小RF积分 | 当前余额、累计获得、累计消耗、最近流水记录 |
| 兑换记录 | 该会员的兑换订单列表（简略） |
| 优惠券 | 该会员持有的券列表 |
| 操作区 | 编辑信息、录入积分、积分更正 |

## 交互说明

- "编辑信息" → 弹窗修改会员基本资料
- "录入积分" → 跳转 P05 积分录入页
- "积分更正" → 弹窗选择积分类型、输入更正值和原因
- 积分流水、兑换记录点击 "查看全部" 可展开完整列表
