# P15 用户管理页

**所属模块**：系统管理

**可见权限**：仅超管

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 搜索栏 | 按姓名、手机号搜索 |
| 筛选条件 | 角色（全部/管理员/普通用户） |
| 用户表格 | 所有通过小程序注册的用户列表 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 姓名 | 用户姓名 |
| 手机号 | 绑定手机号 |
| 角色 | 普通用户 / 管理员 / 超管 |
| 注册时间 | 小程序首次登录时间 |
| 最后登录 | 最近一次登录时间 |
| 操作 | 设为管理员 / 取消管理员 |

## 交互说明

- "设为管理员" → 弹窗设置该用户的PC后台初始登录密码 → 确认后角色变为管理员
- "取消管理员" → 二次确认后降为普通用户，不可再登录PC后台
- 超管账号在列表中展示但不可操作（不可取消超管）
- 此页面仅超管可见，管理员无法进入
