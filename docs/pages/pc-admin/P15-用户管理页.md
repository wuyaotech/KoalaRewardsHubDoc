# P15 用户管理页

**所属模块**：系统管理

**可见权限**：仅超管

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 搜索栏 | 按手机号搜索 |
| 筛选条件 | 角色下拉（全部 / 普通用户 / 管理员 / 超级管理员） |
| 用户表格 | 所有用户列表，分页（默认每页 20 条，可切换） |

**表格字段**：

| 列 | 说明 |
|------|------|
| 姓名 | 姓 + 名拼接，无则显示"—" |
| 手机号 | 绑定手机号，无则显示"—" |
| 角色 | Tag 展示：超级管理员（红色）/ 管理员（蓝色）/ 普通用户（默认） |
| 注册时间 | 格式 YYYY-MM-DD HH:mm |
| 最后登录 | 格式 YYYY-MM-DD HH:mm，无则显示"—" |
| 操作 | 根据角色不同展示不同操作按钮 |

**操作列逻辑**：

| 目标角色 | 可用操作 |
|----------|----------|
| 超级管理员 | 不可操作（灰色文字提示"超管不可操作"） |
| 管理员 | 「重置密码」「取消管理员」 |
| 普通用户 | 「设为管理员」 |

## 弹窗：设为管理员

- 提示文案："设置该用户为管理员后，可使用手机号 + 密码登录 PC 后台。"
- 表单字段：初始登录密码（必填，至少 6 位）+ 确认密码
- 确认后角色变为管理员

## 弹窗：重置密码

- 提示文案："重置该管理员的登录密码，重置后需通知对方使用新密码登录。"
- 表单字段：新密码（必填，至少 6 位）+ 确认密码
- 确认后密码被重置，操作记录到操作日志

## 弹窗：取消管理员

- 二次确认弹窗，内容："确定要取消 XXX 的管理员权限吗？取消后将无法登录 PC 后台。"
- 确认后降为普通用户，密码清空，不可再登录 PC 后台

## 交互说明

- 此页面仅超管可见，管理员无法进入（侧边栏不展示该菜单项）
- 列表支持刷新按钮

## 关联接口

| 接口 | 用途 |
|------|------|
| `POST /admin/users/list` | 分页查询用户列表 |
| `POST /admin/users/set-admin` | 设为管理员 |
| `POST /admin/users/remove-admin` | 取消管理员 |
| `POST /admin/users/reset-password` | 重置管理员密码 |
