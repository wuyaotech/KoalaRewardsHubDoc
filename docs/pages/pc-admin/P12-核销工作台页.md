# P12 核销工作台页

**所属模块**：兑换交易（含核销）

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 扫码区 | 调用摄像头扫描用户出示的核销码二维码 |
| 手动输码 | 输入框，手动输入核销数字码 |
| 核销信息确认 | 扫码/输码后展示订单信息 |
| 核销按钮 | "确认核销" |
| 近期核销记录 | 当前工作人员今日的核销记录列表 |

**核销信息确认展示**：

| 字段 | 说明 |
|------|------|
| 会员姓名 | 兑换人 |
| 商品名称 | 兑换的商品 |
| 兑换数量 | 数量 |
| 花费积分 | 扣减的积分数 |
| 订单状态 | 当前状态 |

## 交互说明

- 扫码成功 → 自动填充并展示订单信息
- 确认核销 → 订单状态变为 "已完成"，用户端同步更新
- 核销码无效/已核销/已过期（10天后）→ 提示错误原因
- 支持在 Pad / 手机浏览器上使用（响应式）
- 每个兑换订单对应一个核销码（二维码 + 8位数字码）
- 不支持撤销核销，核销即完成
