# P09 商品编辑页

**所属模块**：商品管理

---

## 主要内容

| 字段 | 类型 | 说明 |
|------|------|------|
| 积分类型 | 单选 | 大RF商品 / 小RF商品（创建后不可更改） |
| 商品名称 | 文本输入 | 商品标题 |
| 商品图片 | 图片上传 | 支持多张，首张为封面 |
| 商品分类 | 下拉选择 | 选择所属分类 |
| 所需积分 | 数字输入 | 商品统一兑换积分，同一商品下所有 SKU 一致 |
| 限购规则 | 数字输入 | 每人每月限兑数量（0=不限） |
| 商品描述 | 富文本 | 商品详细介绍、规格参数 |
| 规格组配置 | 规格编辑器 | 可添加 0~N 个规格组，为每组勾选规格值，并设置展示顺序 |
| 主规格组 | 单选设置 | 多个规格组时，最多指定 1 个为主规格组，用于规格图片展示 |
| 主规格组规格图 | 图片上传 | 仅主规格组下各规格值可上传对应图片；未上传时沿用商品默认图 |
| SKU 库存 | 自动生成表格 | 系统按规格组合自动生成 SKU；每个 SKU 仅维护库存数量 |

| 操作 | 说明 |
|------|------|
| 保存草稿 | 保存为草稿状态 |
| 保存到仓库 | 保存为仓库（待上架）状态 |
| 保存并上架 | 保存并直接上架 |

## 交互说明

- 编辑模式下回显已有数据
- 修改规格组或规格值后，系统自动重建 SKU 组合，并要求维护各 SKU 库存
- 无规格商品自动生成默认 SKU，后台仅需维护默认 SKU 库存
- 主规格组最多只能设置 1 个；切换主规格组后，对应规格值图片需重新确认
- 必填字段校验：商品名称、积分类型、商品图片、所需积分；有规格时各 SKU 库存必填，无规格时默认 SKU 库存必填
- 保存成功 → 返回 P08 商品列表页
