# P09 商品编辑页

**所属模块**：商品管理

---

## 主要内容

| 字段 | 类型 | 说明 |
|------|------|------|
| 积分类型 | 单选 | 大RF商品 / 小RF商品（创建后不可更改） |
| 商品名称 | 文本输入 | 商品标题 |
| 商品图片 | 图片上传 | 支持多张，首张为封面 |
| 商品分类 | 下拉选择 | 选择所属分类 |
| 所需积分 | 数字输入 | 兑换该商品需要的积分数 |
| 库存数量 | 数字输入 | 可兑换总数 |
| 限购规则 | 数字输入 | 每人每月限兑数量（0=不限） |
| 商品描述 | 富文本 | 商品详细介绍、规格参数 |

| 操作 | 说明 |
|------|------|
| 保存草稿 | 保存为草稿状态 |
| 保存到仓库 | 保存为仓库（待上架）状态 |
| 保存并上架 | 保存并直接上架 |

## 交互说明

- 编辑模式下回显已有数据
- 必填字段校验：商品名称、积分类型、所需积分、库存
- 保存成功 → 返回 P08 商品列表页
