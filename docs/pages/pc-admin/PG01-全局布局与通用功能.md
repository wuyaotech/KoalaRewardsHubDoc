## 右上角用户下拉菜单

| 选项 | 说明 |
|------|------|
| 角色标签 | 展示"超级管理员"或"管理员"（灰色禁用样式，仅展示） |
| 修改密码 | 打开修改密码弹窗 |
| 退出登录 | 清除 token 和用户信息，跳转到登录页 |

用户名展示规则：姓 + 名拼接，无姓名则展示手机号，兜底展示"管理员"。

## 修改密码弹窗

**入口**：右上角用户下拉菜单 → 修改密码

**弹窗内容**：

| 区域 | 说明 |
|------|------|
| 提示文案 | "通过手机号 138****1234 接收验证码来修改密码"（展示当前账号手机号） |
| 验证码 | 6 位，输入框右侧「发送验证码」按钮，60 秒倒计时 |
| 新密码 | 必填，至少 6 位 |
| 确认密码 | 必填，需与新密码一致 |
| 操作按钮 | 「确认修改」/「取消」 |

**交互说明**：
- 修改成功 → toast 提示"密码修改成功"，自动关闭弹窗
- 验证码错误 → toast 提示错误信息
- 两次密码不一致 → 表单校验提示

**关联接口**：

| 接口 | 用途 |
|------|------|
| `POST /auth/send-code` | 发送短信验证码 |
| `POST /auth/admin/change-password` | 提交修改密码 |

## 登录态保护

- 未登录（无 token）→ 自动重定向到 P01 登录页
- 接口返回 401 → 清除登录信息，跳转到登录页
