# P14 数据统计页

**所属模块**：数据统计

---

## 主要内容

页面分为多个统计板块，可用 Tab 或卡片式布局：

### 兑换汇总

| 内容 | 说明 |
|------|------|
| 时间范围选择 | 日/周/月/自定义 |
| 兑换总量 | 时间段内的兑换总笔数、总积分消耗 |
| 按商品排行 | 热门兑换商品 TOP N |
| 趋势图 | 兑换量随时间变化的折线图 |

### 会员概览

| 内容 | 说明 |
|------|------|
| 总会员数 | 注册会员总量 |
| 等级分布 | 各等级会员数量饼图 |
| 新增趋势 | 会员注册趋势 |

### 积分概览

| 内容 | 说明 |
|------|------|
| 积分发放总量 | 大RF + 小RF 分别统计 |
| 积分消耗总量 | 兑换消耗的积分 |
| 即将清零总量 | 未来N个月内将过期的积分 |

### 商品分析

| 内容 | 说明 |
|------|------|
| 热门商品 | 兑换量排名 |
| 库存预警 | 库存低于阈值的商品列表 |

## 交互说明

- 各板块支持时间范围筛选
- 图表支持切换大RF/小RF维度
