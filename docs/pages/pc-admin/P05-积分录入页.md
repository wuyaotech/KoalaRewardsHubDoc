# P05 积分录入页

**所属模块**：积分账户

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 会员选择 | 搜索并选择目标会员（手机号/姓名） |
| 积分类型 | 单选：大RF积分 / 小RF积分 |
| 录入信息 | 根据积分类型展示不同表单 |

**大RF积分录入表单**：

| 字段 | 说明 |
|------|------|
| 消费金额 | 输入消费金额（系统自动换算积分，1元=1积分） |
| 积分数 | 自动计算，可手动微调 |
| 备注 | 录入原因/备注 |

**小RF积分录入表单**：

| 字段 | 说明 |
|------|------|
| 小朋友 | 选择该会员绑定的小朋友 |
| 积分数 | 手动输入积分数 |
| 备注 | 录入原因（如 "XX课程表现优秀"） |

| 操作 | 说明 |
|------|------|
| 确认录入 | 提交并记录操作日志 |

## 交互说明

- 从会员列表/详情页跳转时，自动预填会员信息
- 录入成功 → 提示成功，展示录入结果
- 支持连续录入模式（录完一个不跳走，继续录下一个）
