# P18 积分规则配置页

**所属模块**：系统管理

---

## 主要内容

| 配置项 | 说明 |
|------|------|
| 大RF积分比例 | 消费金额与积分的换算比例（默认 1元=1积分） |
| 大RF积分有效期 | 积分有效期规则（如 自然年清零 / 获取后N个月过期） |
| 小RF积分有效期 | 小RF积分的有效期规则 |
| 清零提醒时间 | 清零前多久开始提醒（如 3个月/2个月/1个月） |
| 其他规则 | 预留扩展配置项 |

## 交互说明

- 修改规则后需二次确认
- 规则变更记入操作日志
- 新规则的生效范围需明确：仅对新积分生效 还是 追溯历史积分（待确认）
