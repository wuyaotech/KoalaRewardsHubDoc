# P11 兑换记录页

**所属模块**：兑换交易

---

## 主要内容

| 区域 | 内容 |
|------|------|
| 筛选条件 | 积分类型、会员姓名/手机号、商品名称、时间范围、订单状态 |
| 兑换订单表格 | 所有兑换记录 |

**表格字段**：

| 列 | 说明 |
|------|------|
| 订单号 | 唯一标识 |
| 会员姓名 | 兑换人 |
| 手机号 | 会员手机号 |
| 积分类型 | 大RF / 小RF |
| 商品名称 | 兑换的商品 |
| 数量 | 兑换数量 |
| 花费积分 | 扣减的积分数 |
| 兑换时间 | 下单时间 |
| 订单状态 | 待核销 / 已完成 / 已过期 |
| 操作 | 查看详情、去核销（仅待核销状态） |

## 交互说明

- "去核销" → 跳转 P12 核销工作台页（预填核销码）
- "查看详情" → 弹窗展示订单完整信息
- 支持导出 Excel（待确认）
