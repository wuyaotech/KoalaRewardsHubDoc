# KoalaRewardsHub 需求文档

> 积分小程序 + PC 管理后台 需求规格说明

---

## 项目概述

为体育俱乐部打造的会员积分系统，包含：

- **微信小程序**：会员查积分、逛商城、兑换商品、查核销码
- **PC 管理后台**：工作人员录积分、管商品、核销、看数据

---
