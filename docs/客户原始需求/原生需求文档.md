RF小程序后台端需求
1）记录会员的基本信息，输入方式为用户主动授权填写（但后台可以做手动更改）
2）后台人员手动为每个会员登入积分信息，输入错误时可更改
3）用户兑换好商品后，后台自动扣除积分并落到对应的会员等级里
4）会员积分有效期为一年（按注册时间开始算起），每年清零，等我清零时间提前三个月、两个月、一个月弹出系统提示。
5）后台可以随时查看每个临近积分清零的会员以便于人工提示。
6）后台可以上架产品（产品图片、名称、参数、兑换价格）、下架产品，也可以把产品放在待发布仓库里等待一键上架或收入仓库。
7）兑换实物产品需要生成一个核销按钮🔘，用户找工作人员领取后，后台点击已领取，同时用户界面也应显示已兑换。
8）每个用户的兑换历史在后台都可见，也可以集中不按用户看加总兑换数据
9）后台可以自动计算每个用户可兑换的产品库存（比如指定商品每月限量三个），也可以手动控制
10）大ＲＦ积分是大人使用的 ，小rf是给8岁以下小孩的，可以和父母绑定在同一个系统里，积分的唯一来源是后台手动输入，没有会员等级，只有一个积分总数，和大RF积分不互通
11）后台可以设置优惠券/代金券，核销靠后台手动点击（已核销）
