# 测试：M01-用户与认证

> 对应需求：[M01-用户与认证](../modules/M01-用户与认证.md)

## 一、小程序登录与注册

### 1.1 微信授权登录

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 首次打开小程序 | 打开小程序，点击微信一键登录 | 成功获取 openid，创建登录账号，进入手机号绑定流程 | |
| 2 | 拒绝授权后重试 | 首次拒绝微信授权 → 再次点击登录 | 可以重新弹出授权弹窗，允许后正常登录 | |
| 3 | 已登录用户再次打开 | 关闭小程序后重新打开 | 自动恢复登录态，无需重新授权 | |

### 1.2 手机号绑定与自动注册

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 新用户绑定手机号（无线下档案） | 微信授权后 → 绑定手机号 | 自动创建会员档案（来源"注册转会员"）+ 自动创建大RF积分账户 | |
| 2 | 新用户绑定手机号（有线下档案） | 后台已建档手机号 123 → 用户用 123 绑定 | 自动匹配已有会员档案，状态从"未激活"变为"已激活" | |
| 3 | 手机号已被其他账号绑定 | 用户 A 已绑定 123 → 用户 B 也尝试绑定 123 | 提示冲突，不允许重复绑定 | |
| 4 | 仅授权未绑定手机号 | 微信授权后不绑定手机号，直接浏览 | 可以浏览部分页面，但无会员身份，不能兑换商品 | |

### 1.3 会员档案激活

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 线下建档后首次登录 | 后台录入会员 → 会员首次打开小程序绑定同一手机号 | 自动匹配并激活，会员数据（积分、等级等）完整保留 | |
| 2 | 线下建档的会员有积分 | 后台建档后录入了积分 → 会员首次激活 | 激活后可在小程序看到已有积分余额 | |

## 二、PC后台登录

### 2.1 密码登录

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 管理员正常登录 | 输入正确手机号 + 密码 | 登录成功，进入工作台 | |
| 2 | 密码错误 | 输入正确手机号 + 错误密码 | 提示密码错误 | |
| 3 | 手机号不存在 | 输入未注册手机号 | 提示账号不存在或无权限 | |
| 4 | 普通会员尝试登录后台 | 用普通会员的手机号+密码登录 | 提示无权限，不允许登录 | |

### 2.2 验证码登录

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 管理员验证码登录 | 输入手机号 → 获取验证码 → 输入正确验证码 | 登录成功 | |
| 2 | 验证码错误 | 输入错误验证码 | 提示验证码错误 | |
| 3 | 验证码过期 | 等验证码过期后再输入 | 提示验证码已过期 | |

## 三、密码管理

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 管理员自行修改密码 | 右上角菜单 → 修改密码 → 手机验证码验证 → 输入新密码 | 密码修改成功，旧密码失效 | |
| 2 | 超管重置管理员密码 | 用户管理页 → 选择管理员 → 重置密码 | 管理员密码被重置，操作记入日志 | |
| 3 | 修改密码后旧密码登录 | 修改密码成功后 → 用旧密码登录 | 登录失败 | |
| 4 | 修改密码后新密码登录 | 修改密码成功后 → 用新密码登录 | 登录成功 | |

## 四、权限边界

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 普通会员访问后台接口 | 用普通会员 token 调用 admin 接口 | 返回 403 无权限 | |
| 2 | 管理员尝试重置他人密码 | 非超管的管理员尝试重置密码操作 | 被拒绝，仅超管可操作 | |
| 3 | token 过期后访问 | 使用过期的 token 调用接口 | 返回 401 未认证 | |
| 4 | 未登录访问需登录接口 | 不带 token 调用需登录的接口 | 返回 401 | |
