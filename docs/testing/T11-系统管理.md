# 测试：M11-系统管理

> 对应需求：[M11-系统管理](../modules/M11-系统管理.md)

## 一、用户列表管理（PC端）

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 查看用户列表 | 打开用户管理页 | 展示所有注册用户，含角色标识 | |

## 二、管理员角色管理

### 2.1 设置管理员（超管操作）

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 超管设置管理员 | 用户列表 → 选择普通用户 → 设置为管理员 → 设置初始密码 | 用户角色变为管理员，可登录PC后台 | |
| 2 | 设置后可登录 | 新管理员用初始密码登录PC后台 | 登录成功 | |
| 3 | 操作日志记录 | 设置管理员后查看操作日志 | 记录：操作人、被设置人姓名/手机、操作时间 | |

### 2.2 取消管理员（超管操作）

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 超管取消管理员 | 用户列表 → 选择管理员 → 取消管理员 | 用户角色降为普通用户 | |
| 2 | 取消后不可登录后台 | 被取消管理员尝试登录PC后台 | 登录失败，提示无权限 | |
| 3 | 操作日志记录 | 取消管理员后查看日志 | 记录完整操作信息 | |

### 2.3 权限边界

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 管理员尝试设置管理员 | 非超管的管理员执行设置操作 | 被拒绝 | |
| 2 | 管理员尝试取消管理员 | 非超管的管理员执行取消操作 | 被拒绝 | |
| 3 | 超管不可被取消 | 尝试取消超管角色 | 被拒绝（超管只能数据库修改） | |

## 三、操作日志

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 积分录入日志 | 录入大RF积分后查看日志 | 记录：POINTS_GRANT_BIG、会员姓名/手机、积分数、备注 | |
| 2 | 小RF积分录入日志 | 录入小RF积分后查看日志 | 记录：POINTS_GRANT_SMALL、含小朋友信息 | |
| 3 | 积分更正日志 | 积分更正后查看日志 | 记录：POINTS_CORRECT、调整值、原因、调整后余额 | |
| 4 | 核销操作日志 | 核销订单后查看日志 | 记录：VERIFY_ORDER、订单UUID、核销码 | |
| 5 | 商品上架日志 | 商品上架后查看日志 | 记录：PRODUCT_ON_SALE、商品名称、状态变化 | |
| 6 | 商品下架日志 | 商品下架后查看日志 | 记录：PRODUCT_OFF_SALE、商品名称 | |
| 7 | 设置管理员日志 | 设置管理员后查看日志 | 记录：USER_SET_ADMIN、被设置人信息 | |
| 8 | 取消管理员日志 | 取消管理员后查看日志 | 记录：USER_REMOVE_ADMIN、被取消人信息 | |
| 9 | 日志不可删除 | 尝试删除日志 | 操作日志永久保留，无删除功能 | |
| 10 | 日志操作人 | 任意日志记录 | operatorUuid 正确，对应当前登录管理员 | |
| 11 | 日志 detail 内容 | 查看 detail 字段 | JSON 格式，包含操作关键快照信息 | |

## 四、积分规则配置（PC端）

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 查看当前配置 | 打开积分规则配置页 | 展示当前的积分比例、有效期规则、清零提醒时间 | |
| 2 | 修改积分比例 | 将大RF积分比例从 1:1 改为 2:1 | 保存成功，仅对后续新录入生效 | |
| 3 | 修改有效期规则 | 修改积分有效期 | 保存成功 | |
| 4 | 不追溯历史 | 修改规则后查看已有积分 | 已有积分不受影响 | |
| 5 | 仅超管可配置 | 普通管理员尝试修改积分规则 | 应被拒绝（确认是否超管专属） | |
