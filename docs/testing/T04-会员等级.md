# 测试：M04-会员等级

> 对应需求：[M04-会员等级](../modules/M04-会员等级.md)

## 一、等级计算与升级

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 新会员初始等级 | 用户注册成为会员 | 初始等级 Lv.1 考拉（累计积分 0） | |
| 2 | 升到 Lv.2 | 累计录入大RF积分达到 6000 | 自动升级到 Lv.2 小熊 | |
| 3 | 升到 Lv.3 | 累计录入大RF积分达到 12000 | 自动升级到 Lv.3 小马 | |
| 4 | 升到 Lv.4 | 累计录入大RF积分达到 30000 | 自动升级到 Lv.4 海豚 | |
| 5 | 升到 Lv.5 | 累计录入大RF积分达到 50000 | 自动升级到 Lv.5 花豹 | |
| 6 | 恰好在门槛上 | 累计大RF积分恰好为 6000 | 升到 Lv.2（门槛值包含在上一级） | |
| 7 | 跳级升级 | 一次性录入大RF积分 50000（从 0 直接达到 Lv.5 门槛） | 直接升到 Lv.5，不需要逐级经过 | |

## 二、等级不降级

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 积分清零后等级不变 | Lv.3 会员 → 注册周年积分清零 | 等级维持 Lv.3，不降级 | |
| 2 | 积分消费后等级不变 | Lv.4 会员 → 兑换商品消费大量积分 | 等级维持 Lv.4（等级按累计获得积分算，非当前余额） | |

---

## 三、积分更正对等级的影响

| # | 场景 | 操作步骤 | 预期结果 | 通过 |
|---|------|----------|----------|------|
| 1 | 增加更正触发升级 | 累计 5900 → 更正增加 200 → 累计 6100 | 触发等级重算，升到 Lv.2 | |
| 2 | 减少更正影响等级 | 累计 6100 → 更正减少 200 → 累计 5900 | 等级是否降级需确认业务规则（更正是否算降级？） | |

## 四、小RF积分不影响等级
