# 模块04：会员等级

**职责**：根据大RF累计获得积分自动计算会员等级，管理各等级的权益配置

## 能力清单

| # | 能力 | 端 | 描述 |
|---|------|---|------|
| 1 | 等级展示 | 小程序 | 展示当前等级名称、等级图标、距下一级的积分差距 |
| 2 | 等级权益查看 | 小程序 | 查看各等级对应的权益列表 |
| 3 | 自动升级 | 系统 | 大RF累计获得积分达到门槛时自动升级，清零后不降级 |
| 4 | 等级规则配置 | PC | 配置等级名称、积分门槛、权益内容 |

---

## 业务规则

### 等级体系

大RF**累计获得积分**达到下表门槛即升级到该等级；等级由低到高，仅大RF积分参与计算。

| 等级 | 名称 | 积分区间（大RF累计） | 门槛 minTotalPoints |
|------|------|---------------------|---------------------|
| Lv.1 | 考拉 | 0 - 5,999 | 0 |
| Lv.2 | 小熊 | 6,000 - 11,999 | 6,000 |
| Lv.3 | 小马 | 12,000 - 29,999 | 12,000 |
| Lv.4 | 海豚 | 30,000 - 49,999 | 30,000 |
| Lv.5 | 花豹 | 50,000+ | 50,000 |

### 等级计算

1. **仅大RF积分参与**：小RF积分不影响会员等级
2. **计算依据**：按大RF累计获得积分计算等级，不按当前可用积分计算
3. **升级规则**：累计积分达到门槛即升级
4. **清零规则**：积分清零后等级不降级，维持现有等级
5. **系统自动计算距下一级所需的累计积分差距**

### 等级权益

- 权益内容由后台配置，仅作展示用途
- 实际权益在线下兑现（如生日礼、停车优惠、课程优惠、购买时直接打折等）
- 每个等级有各自对应的权益列表
